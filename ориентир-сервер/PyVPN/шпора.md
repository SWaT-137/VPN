Запустить: sudo systemctl start pyvpn
Остановить: sudo systemctl stop pyvpn
Перезапустить (после изменения кода): sudo systemctl restart pyvpn
Узнать статус (работает ли): sudo systemctl status pyvpn
Посмотреть логи в реальном времени: sudo journalctl -u pyvpn -f
Редактировать код: sudo nano /opt/VnP/server.py
